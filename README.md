<a name="english"></a>
## :gb: English

### Overview

An interactive dashboard to explore, analyze, and visualize data with a Flask API backend. Upload datasets and generate insights through dynamic charts and analytics.

### Features

- **Interactive Charts** - Dynamic visualizations with Plotly
- **Data Upload** - CSV and Excel file support
- **Flask API** - RESTful backend for data processing
- **Real-Time Analytics** - Live data exploration
- **Export Reports** - Download analysis results

### Tech Stack

| Component | Technology |
|-----------|------------|
| Dashboard | Streamlit |
| Backend | Flask |
| Visualization | Plotly, Matplotlib |
| Data Processing | Pandas, NumPy |

### Quick Start

```bash
git clone https://github.com/fracabu/dashboard-data-visualization.git
cd dashboard-data-visualization

pip install -r requirements.txt

# Start Flask API
python api.py

# Start Dashboard (new terminal)
streamlit run app.py
```
